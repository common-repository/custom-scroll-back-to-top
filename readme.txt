== Description ==
Custom Scroll Back to Top allows you to scroll back up to the top of a website page by clicking your customized icon or text link.

== Installation ==
1. Plugins > Add New > Search `Custom Scroll Back to Top`
2. Install Now > Activate


**Important**

Always try at your local website first.
